JMR API 介绍

JMR API 是一个多模型 API 接入平台，目标是让顶级模型在国内场景也能稳定、低门槛使用。

  • 支持主流模型与多模态输入（文本 / 图片）
  • 强调稳定可用与透明计价
  • 可对接 OpenClaw、OpenCode、Codex、Chatbox、Cherry Studio 等客户端

你可以在这里做什么

  • 快速完成 OpenClaw 部署与 API 改接
  • 查看 macOS / Linux 的最短可用路径
  • 按排障清单定位常见问题
