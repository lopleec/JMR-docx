排障与安全

最常见问题

1) 识图失败

  • 确认模型本身支持 image 输入
  • 确认 `imageModel` 指向可用模型
  • 本地附件先转存到允许目录（如 workspace）

2) iMessage 只能收到占位符

  • 开启 `includeAttachments`
  • 设置 `attachmentRoots`
  • 重启 gateway 并复测

3) 定时任务偶发失败

  • 增加超时上限，避免网络抖动导致中断
  • 记录失败日志并做自动重试
  • 关键任务建议 fallback 数据源

安全基线

  • 最小权限：只开必须的工具和通道
  • API key 只放服务端，不进前端代码
  • 定期执行安全审计和版本更新
  • 公网暴露时务必加鉴权与来源限制
