本机部署（macOS/Linux）

准备项

  • Node.js 与 OpenClaw 已安装
  • 至少一个模型可用（文本）
  • 如需识图，模型必须声明支持 image 输入

启动顺序

openclaw gateway start openclaw status --deep

iMessage 特别说明

如果你用 iMessage，建议启用附件抓取：

{ "channels": { "imessage": { "enabled": true, "includeAttachments": true, "attachmentRoots": ["/Users/*/Library/Messages/Attachments"] } } }

然后重启 gateway：

openclaw gateway restart

图片识别实践建议

本地媒体读取有安全白名单，建议统一将附件转存到 workspace 后再识别。

# 示例思路 # 1) 附件复制到 ~/.openclaw/workspace # 2) 再调用 image 工具识别

本机部署优缺点

  • 优点：调试快，修改配置后反馈快
  • 缺点：机器关机或网络变化会中断服务
